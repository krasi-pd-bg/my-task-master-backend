# TaskMaster Backend

Backend for the TaskMaster React Native app.

## Tech Stack
- json-server
- json-server-auth

## API URL
https://my-task-master-backend.onrender.com

## API Endpoints
- `POST /register`
- `POST /login`
- `GET /tasks?userId=...`
- `POST /tasks`
- `PUT /tasks/:id`
- `PATCH /tasks/:id`
- `DELETE /tasks/:id`
- `GET /categories?userId=...`
- `POST /categories`
- `DELETE /categories/:id`
- `GET /users/:id`
